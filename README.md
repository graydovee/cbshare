# share clipboard
* share clipboard from different machine
* 在多个机器间共享剪贴板
* develop base on [clipboard](https://github.com/atotto/clipboard)

## build 
```bash
make build
```

# run
use `cbshare -h` get more info
## server
```bash
cbshare server -p <port>
```

## client
```bash
cbshare client -a <addr>:<port>
```
